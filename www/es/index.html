Palabras de Sabiduría
¿Qué problema resuelve este proyecto?
Los proyectos de desarrollo de software necesitan mucho "papeleo" en forma de documentos de requerimientosm documentos de diseño, planes para pruebas, calendarios, listas de pendientes, notas de versión, etc. Parecería que todos creamos los documentos de una página en blanco, de los documentos utilizados en el último proyecto, o de una de las carísimas librerías de plantillas de propietario para ingeniería de software. Para aquellos que empiezan con una página en blanco, puede ser demasiado trabajo y es fácil olvidar partes importantes. Esta no es una base muy confiable para los proyectos profesionales de ingeniería.
¿Cuál es el objetivo de este proyecto?
ReadySET es un proyecto open-source para producir y mantener una librería de plantillas reutilizables para ingeniería de software. Estas plantillas proveen un punto partida para los documentos utilizados en proyectos de desarrollo de software. Utilizando buenas plantillas puede ayudar a los desarrolladores a trabajar más rápido, pero también ayudarán a evitar discusiones y a evitar pasar por alto problemas importantes.
¿Cuáles son algunas de las características más importantes que definen a este producto?
  • Guías de alta calidad, texto de ejemplo y listas de pendientes.
  • Utiliza tecnologías web sencillas: XHTML puro y CSS.
  • Plantillas para muchos documentos comunes de ingeniería de software.
¿Qué hace a este producto diferente de otros?
Este es un proyecto open-source que se le invita a usar de forma gratuita y a ayudar a mejorar. Los paquetes existentes de plantillas de ingeniería de software son muy costosas y están limitadas por la autoría de solo unas cuantas personas, por relaciones cliente-vendedor o por un conjunto de herramientas ofrecidas por un distrbuidor en particular.
Estas plantillas son XHTML puro con CSS, no un formato de archivo propietario. Lo que las hace más fácil de editar y de dar seguimiento a los cambios usando herramientas libres disponibles y sistemas de control de versiones. Las plantillas están diseñadas para ser siempre usadas en un ambiente web; utilizan hiperligas para evitar la duplicación de información.
Las pantillas no esán saturadas de información sobre autoría personal o el historial de cambios del documento. Se asume que los desarrolladores de software profesionales mantendrán todos los documentos del proyecto en sistemas de control de versiones que proveerán a las plantillas con esas capacidades.
Estas plantillas no son universales y no intentan proveer guías prescriptivas en el proceso general de desarrollo. Estamos desarrollando una amplia librearía de módulos de plantillas para muchos propósitos y procesos. Las plantillas pueden ser llenadas en la secuencia sugerida o en cualquier secuencia que se ajuste a sus procesos existentes. Estas pueden ser fácilmente personalizadas con cualquier editor de texto o HTML.

Alcance

¿Cuál es el alcance de este proyecto?
Construir plantillas para documentos comunes de ingeniería de software inspiradas en nuestra propia experiencia.
¿Cuáles son las suposiciones de alo nivel o las reglas básicas para este proyecto?
Asumo que el usuario se hace completamente responsable por el contenido de los documentos de sus proyectos actuales. Las plantillas son meramente puntos de inciio y guías de bajo nivel.
¿Qué es lo que no debe de hacer?
Este proyecto no intenta proveer con herramientas poderosas para reorganizar las plantillas, mapearlas a un software dado o proceso de desarrollo, o generar plantillas desde un modelo de proceso básico. Este proyecto no incluye ningún código de aplicación para ninguna herramienta, los usuarios simplemente utilizarán editores de texto para llenar o personalizar las plantillas.
¿Es este proyecto parte de un esfuerzo más grande?
Sí, es parte de la misión de Tigris.org de promover la ingeniería de software open-source. Es también el primer producto en una línea de productos que ayudarán a proveer un mejor soporte a los desarrolladores profesionales de software. Para más información, vea ReadySET Pro.

Estado

¿Cuál es el estado de este proyecto?
Estas plantillas están basadas en plantillas usadas originalmente para enseñar ingeniería de software en un curso de proyectos en una universidad. Estas plantillas están siendo mojradas, expandidas y usadas de una forma más amplia por profesionales en la industria.
El conjunto de plantillas está casi completo y listo para usarse en proyectos reales. Ud. puede descargar las últimas versiones (en inglés). Todas las sugerencias son bienvenidas.
Para las últimas noticias, vea las Noticias del Proyecto.

Aplicabilidad

¿Quién debe usar este producto?
ReadySET está orientado a ingenieros de software que desean que sus proyectos pueden fluir de una forma más sencilla y profesional. ReadySET puede ser usado por cualquier persona que sea capaz de utilizar un editor HTML o editar HTML en un editor de texto.
¿Cómo pueden los usuarios empezar a trabajar con las plantillas?
  1. Navegue por el conjunto de plantillas brevemente
  2. Descargue las plantillas y desempáquelas
  3. Edite las plantillas para adecuarlas a las necesidades de su proyecto
  4. Coloque las plantillas en un servidor web donde todas las personas involucradas en el proyecto puedan verlas
  5. Edite las plantillas para llenarlas con información más detallada
    • Use un editor de texto o un editor HTML. Revise nuestra lista de herramientas recomendadas. (Puede usar Word, pero no se recomienda.)
    • Siga las instrucciones que aparecen en las cajas amarillas o "notas adheridas"
    • Reemplace el texto en MAYÚSCULAS con el texto que describe su proyecto
    • "Esculpa" el texto que no aplique a su proyecto
    • Añada texto, diagramas o ligas según considere necesario
  6. Use las listas de pendientes para descubrir errores comunes y mejorar la calidad de sus documentos.
  7. Use las páginas de notas de sabiduría (words-of-wisdom) para ayudar a mejorar el documento o profundizar su comprensión en asuntos clave.
  8. Opcionalmente, puede cambiar las funtes y colores en el archivo css/inst.css.
  9. Si tiene preguntas o comentarios sobre las plantillas, por favor lea la sección de Preguntas Frecuentes o envie un correo electrónico a dev@readyset.tigris.org.
