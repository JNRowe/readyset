TIPs:
  • Add HTML anchors on your terms so that other documents can link to the definition of specific terms. That will also allow links to other terms in the definitions.
  • If there is any question about the meaning of a term, note it here. If someone (e.g., the customer) gave you a definition to use, note that here too. If something is best defined by using a hyperlink to another document or website, include a hyperlink in the definition.
  • If a term was used, but is no longer going to be used, you should keep it in the list and mark it as "deprecated".
  • Just define the special terms used in this project, or ones that a new team member would not know. Don't start defining hundreds of textbook terms. You can always find these at dictionary.com or Google.
  • Don't waste time alphabetizing, just group terms by first letter. Users can use Control-F to "Find..." the terms when needed.
